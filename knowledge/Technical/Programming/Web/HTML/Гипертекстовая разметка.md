# Гипертекстовая разметка
тег - специальная конструкция, основной элемент создания разметки
HTML = Hyper Text Markup Language
## конструкция тегов
![[Pasted image 20260221145320.png]]
атрибуты - более тонкая настройка
![[html-tag-structure.png]]

```html 
<!doctype html> 
``` 
говорит браузеру какую спецификацию html использовать, что здесь начался файл

```html
<html>
</html>
``` 
весь контент сайта должен располагаться внутри этого тега
говорит о том, что здесь непосредственно начался html

```html 
<head>
</head>
``` 
хранятся специальные данные: мета-информация, title, description (содержит описание сайта) 
из всех данных на сайте отображается только title

```html 
<body>
</body>
``` 
располагается весь контент, который виден пользователю

## базовые теги
[сайт со всеми тегами](https://html5book.ru)




strong и em выделяют важное 
голосовые помощники выделяют их интонационно
b и i просто форматирование
q - цитата
cite - автор
```html
<p>
	<q>Исходный код - это ответственность, а не актив!</q>
	<cite>- Эрик Ли</cite>
</p>
```

figure - для группировки связанных элементов: визуального и текстового
```html
<figure>
  <img width="600" src="https://gas-kvas.com/uploads/posts/2023-02/1675489758_gas-kvas-com-p-izobrazheniya-i-kartinki-na-fonovii-risuno-41.jpg" alt="">
  <figcaption>leopard</figcaption>
</figure>
```

a - ссылка

кнопка делает что-то функциональное (какое-то действие)

```html
<table>
    <caption>Название таблицы</caption>
    <thead>
        <tr>
            <th>заголовок 1</th>
            <th>заголовок 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>ячейка 1-1</td>
            <td>ячейка 1-2</td>
        </tr>
        <tr>
            <td>ячейка 2-1</td>
            <td>ячейка 2-2</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td colspan="2">итого</td>
        </tr>
    </tfoot>
</table>
```

в tfoot могут писаться примечания
